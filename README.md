# Fetch and Display API Data

##  About This Project

This project is a simple web application that fetches user data from a public API and displays it on the webpage.

I built this project to practice working with APIs using JavaScript. It helped me understand how to fetch data, convert it into JSON, and display it dynamically using DOM manipulation. I also focused on creating a clean and responsive user interface using HTML and CSS.

##  Features

- Fetches user data from a public API
- Displays user information in a clean layout
- Responsive design for mobile and desktop
- Simple and easy-to-understand code structure

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API



##  Project Structure

```
project-folder/
│── index.html
│── style.css
│── script.js
└── README.md
```

##  What I Learned

- How to use the Fetch API
- Working with JSON data
- DOM manipulation
- Handling asynchronous JavaScript using Promises
- Building responsive web pages

## ▶ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. The user data will be fetched automatically and displayed on the page.
